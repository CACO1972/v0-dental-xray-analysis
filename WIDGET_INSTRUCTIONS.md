# Widget Zero Caries - Instrucciones de Integración

## Para Clínica Miro

### Opción 1: Integración Simple (Recomendado)

Agrega este código HTML donde quieras mostrar el widget:

```html
<div id="zero-caries-widget"></div>
<script src="https://your-domain.vercel.app/api/widget-embed"></script>
```

### Opción 2: IFrame Directo

```html
<iframe 
  src="https://your-domain.vercel.app/widget" 
  width="100%" 
  height="800px" 
  frameborder="0"
  style="border-radius: 16px; box-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.1);"
></iframe>
```

### Opción 3: Enlace Externo

Simplemente enlaza a la aplicación completa:

```html
<a href="https://your-domain.vercel.app" target="_blank">
  Detecta Caries con Zero Caries
</a>
```

## Personalización

El widget se adapta automáticamente al ancho del contenedor. Puedes ajustar la altura según necesites.
